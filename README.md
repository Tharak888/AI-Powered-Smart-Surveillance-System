# AI-Powered Smart Surveillance System

## 📌 Overview
This project is an intelligent surveillance system using:
- Face Recognition (DeepFace / ArcFace)
- Fall Detection (MediaPipe Pose)
- Theft Detection (YOLOv8)

## 🚀 Features
- Real-time face recognition
- Fall detection with alerts
- Multi-factor theft detection
- Zone-based monitoring

## 🛠️ Technologies Used
- Python
- OpenCV
- YOLOv8
- MediaPipe
- DeepFace
